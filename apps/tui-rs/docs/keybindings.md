# Keybindings

Identical to cv-tui-go for a consistent experience across both TUIs.

## Global

| Key | Action |
|-----|--------|
| `q` | Quit (with confirmation if action is running) |
| `Ctrl+C` | Force quit |
| `?` | Toggle help overlay |
| `Tab` | Cycle between panes/views |
| `Esc` | Go back / close modal |
| `1`-`6` | Jump to view: 1=Dashboard, 2=Apps, 3=Kanban, 4=Actions, 5=Stats, 6=Settings |

## Navigation

| Key | Action |
|-----|--------|
| `j` / `Down` | Move down |
| `k` / `Up` | Move up |
| `h` / `Left` | Move left (kanban columns) |
| `l` / `Right` | Move right (kanban columns) |
| `g` | Jump to top |
| `G` | Jump to bottom |
| `Enter` | Select / open |
| `/` | Filter / search |
| `Esc` | Clear filter |

## Applications list

| Key | Action |
|-----|--------|
| `n` | New application |
| `a` | Run apply pipeline |
| `d` | Delete application (with confirmation) |
| `Enter` | Open application detail |
| `/` | Filter by company/position |
| `s` | Cycle sort (date, company, status) |
| `f` | Filter by status |

## Application detail

| Key | Action |
|-----|--------|
| `r` | Run action on this application |
| `e` | Edit meta.yml |
| `o` | Open job URL in browser |
| `y` | Copy application path |
| `t` | Tailor application |
| `v` | Review application |
| `b` | Build application |
| `s` | Score application (ATS) |
| `p` | Prep interview |
| `a` | Run audit on application |
| `Tab` | Cycle between info/files/actions panes |
| `Esc` | Back to applications list |

## Kanban board

| Key | Action |
|-----|--------|
| `h` / `l` | Move between columns |
| `j` / `k` | Move between cards in column |
| `Enter` | Open application detail |
| `m` | Move card to next stage |
| `M` | Move card to specific stage |

## Action runner

| Key | Action |
|-----|--------|
| `r` | Run / re-run action |
| `Ctrl+C` | Cancel running action |
| `y` | Copy output to clipboard |
| `w` | Toggle line wrap |
| `j` / `k` | Scroll output |
| `g` / `G` | Top / bottom of output |

## Stats view

| Key | Action |
|-----|--------|
| `R` | Refresh data |
| `Tab` | Cycle between charts |

## New Application dialog

| Key | Action |
|-----|--------|
| `Tab` | Next field |
| `Enter` | Submit form (on last field) |
| `Esc` | Cancel |

## Audit view

| Key | Action |
|-----|--------|
| `j` / `k` | Navigate applications |
| `Enter` | Run audit on selected |
| `Esc` | Back |
